# Urbanova

Urbanova is a client-server web application for managing e-scooter rentals.

## Tech Stack

Backend:
- Spring Boot
- Maven
- MySQL (Docker)

Frontend:
- Vue 3
- Element Plus

Mobile (Planned for Sprint 3):
- Capacitor

## Architecture

- REST API provided by Spring Boot backend
- Vue-based web client
- MySQL database for persistence

## Sprint 1 Scope

Backlog items:
- ID 1: User registration & login
- ID 4: View hire options and cost
- ID 5: Book an e-scooter
- ID 12: Cancel booking

## Branching Strategy

- `main`: stable and runnable version
- All development must be done in `feature/*` branches
- Pull request required before merging into `main`
- At least 1 review required
- No direct push to `main`
